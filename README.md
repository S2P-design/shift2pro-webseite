# SHIFT2PRO Website

Offizielle Website und Coming-Soon-Seite für den **SHIFT2PRO** Eishockey-Development-Podcast.

## Dateien

| Datei | Beschreibung |
|---|---|
| `index.html` | Hauptseite mit allen Sektionen |
| `coming-soon.html` | Coming-Soon-Seite für den Podcast-Launch |

## Features

- Dark-Mode Corporate Design (Schwarz / Cyan `#0CC0DF`)
- Vollbild-Hero mit Hintergrundbild und Eishockey-Spielfeld-SVG-Overlay
- Cursor Glow Effekt
- Animierter 100+ Folgen Counter
- Trailer-Bereich mit iPhone-Mockup und eingebettetem Video
- Podcast-Plattform-Links (Spotify, Apple Podcasts, YouTube, Amazon Music)
- Responsive (Mobile & Desktop)
- Alle Assets als Base64 eingebettet — keine externen Abhängigkeiten

## Deployment

Die Seiten sind reine HTML-Dateien ohne Build-Schritt. Einfach auf einen Webserver hochladen oder über **GitHub Pages** bereitstellen.

### GitHub Pages

1. Repository auf GitHub erstellen
2. Unter *Settings → Pages* die Branch `main` und Root `/` auswählen
3. `index.html` ist automatisch die Startseite
4. `coming-soon.html` ist erreichbar unter `https://<username>.github.io/<repo>/coming-soon.html`
